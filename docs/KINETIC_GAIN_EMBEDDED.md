# Kinetic Gain Embedded — tie-back

## What this repo demonstrates

This repo demonstrates the **campaign taxonomy governance** primitive — naming changes tied to audience blockers, launch posture, and operator-safe approvals.

## How Kinetic Gain Embedded extends it

Kinetic Gain Embedded is security-first embedded analytics for B2B SaaS.
It extends this primitive to attribution-aware and revenue-aware in-product analytics for growth, campaign, and CRM workflows, wrapped in:

- signed JWT authentication
- semantic-layer row-level security (RLS)
- read-only data connectors
- AST SQL validation
- append-only audit events
